# PROXUMA Canonical Ecosystem

This is the canonical GitHub Pages repository for the PROXUMA family.

## Live routes

- `/PROXUMA/` — canonical ecosystem homepage
- `/PROXUMA/lite/` — Proxuma Lite
- `/PROXUMA/sense/` — Proxuma Sense
- `/PROXUMA/shield/` — Proxuma Shield
- `https://scar197124.github.io/Proxuma_IT/` — Proxuma IT

## Important deployment rule

The root `index.html` must always remain the ecosystem homepage. Do not upload a product app's `index.html` to the repository root.

## Restore map

- Root `index.html` → canonical ecosystem page
- `lite/index.html` → Lite
- `sense/index.html` → Sense
- `shield/index.html` → Shield
- `shared/` → shared styles, UI scripts, QR compatibility, and security helpers

## Upload

Upload the contents of this folder to the `main` branch of the `scar197124/PROXUMA` repository.
