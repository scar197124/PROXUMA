# Proxuma Sense (Anchor Build)

Proxuma Sense is a privacy-first, offline risk analysis tool.

## Features
- Local-only scanning (no API calls)
- Signal-based detection
- Pattern explanation
- Human-readable results

## Usage
Open index.html or sense.html in a browser.

## Privacy
All analysis runs locally. No data leaves your device.
