# PROXUMA v12.23 — Sense Scanner + Shield Zero-State Patch

## Changes

### Proxuma Sense
- Added a persistent **Scanner** navigation button.
- Users can now move from About, Privacy, or Transparency back to the Sense scanner without returning to the canonical homepage.
- Added a small responsive nav override so the added Scanner button remains mobile-friendly.

### Proxuma Shield
- Changed the default Intelligence Core display from a prefilled low-risk state to a neutral no-scan state.
- Default score now shows **0/100**.
- Default status now shows **NO SCAN YET** instead of **LOW RISK**.
- Default signal bars now show **0/100**.
- Clear Inputs now restores the same neutral no-scan state and resets report metadata.

## Preserved
- Canonical PROXUMA homepage.
- Lite / Sense / Shield structure.
- Proxuma IT link.
- Existing scan behavior after a real scan is run.
- Offline-first local analysis behavior.
